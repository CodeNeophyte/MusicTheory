  • *(_:_:)

    Extends Interval by any given octave. For example between C1 and D1, there are 2 halfsteps but between C1 and D2 there are 14 halfsteps. / .M2 * 2 will give you D2 from a C1.

    Declaration

    Swift

    public func *(interval: Interval, octave: Int) -> Interval

    Parameters

    octave

    Return Value

    Returns new interval by calculating halfsteps between target interval and octave.
